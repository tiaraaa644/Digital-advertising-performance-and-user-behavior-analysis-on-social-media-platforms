# 📊 Digital Advertising Performance and User Behavior Analysis

## 📌 Overview
This project analyzes digital advertising performance and user behavior on social media platforms. The goal is to understand engagement, conversion, targeting effectiveness, and campaign efficiency.

---

## 🎯 Objectives
- Identify ads with highest engagement and purchases
- Analyze conversion rate performance
- Evaluate targeting accuracy
- Understand user activity patterns
- Analyze demographic behavior
- Measure campaign efficiency

---

## 🛠️ Tools & Technologies
- SQL
- Data Analysis
- Visualization : Power BI

## 📊 Dataset

The dataset used in this project is Social Media Advertisement Performance from Kaggle. It simulates digital advertising performance and user behavior on social media platforms.

The dataset consists of four main tables:

* **Users**: user demographic data (age, gender, interests, etc.)
* **Campaigns**: campaign information (budget, duration)
* **Ads**: advertisement data and targeting parameters
* **Ad Events**: user interactions (impressions, likes, shares, purchases)

This dataset is used to analyze campaign performance, targeting effectiveness, and user behavior throughout the conversion funnel.

## 🔍 Analysis Performed

2. Conversion Rate Analysis

---

## 📈 Key Insights
- High engagement does not always mean high conversion  
- Majority of traffic comes from non-matching interests  
- Peak user activity occurs at specific times  
- Certain campaigns are efficient despite lower volume  

---

## 💡 Recommendations
- Optimize targeting strategy  
- Focus on high-performing campaigns  
- Improve landing page experience  
- Apply A/B testing for better results
